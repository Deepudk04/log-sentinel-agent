# ERR-003: Caught Exceptions Are Logged Or Propagated

Detects catch or except blocks that swallow exceptions without logging, rethrowing, or returning a controlled generic error path.

Severity: `medium`

Recommended remediation: log unexpected exceptions server-side with useful investigation context, or propagate them to a centralized handler that performs equivalent logging.

Suppression:

```java
// logsentinel-disable-next-line ERR-003
catch (RuntimeException ex) {
}
```
