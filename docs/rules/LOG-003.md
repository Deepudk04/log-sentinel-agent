# LOG-003: Sensitive Data Is Excluded From Logs

Detects logging statements that include sensitive values such as passwords, tokens, API keys, session IDs, cookies, private keys, connection strings, or payment-card-looking data.

Severity: `high`

Recommended remediation: redact, hash, tokenize, or remove sensitive fields before logging. Prefer structured logging with allowlisted safe fields.

Suppression:

```python
# logsentinel-disable-next-line LOG-003
logger.info("password=%s", password)
```
